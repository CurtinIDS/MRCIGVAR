# Installation

Install the development version from GitHub with `pak`:

```r
# install.packages("pak")
pak::pak("CurtinIDS/MRCIGVAR")
```

For local development, clone the repository and either load or install the
package locally:

```r
devtools::load_all()
# or
devtools::install()
```

## Package requirements

`MRCIGVAR` depends on:

- `vars`
- `Matrix`
- `ggplot2`
- `gridExtra`
- `stats`

Useful development dependencies include:

- `knitr`
- `rmarkdown`
- `devtools`
- `roxygen2`
- `testthat`

## Notes for contributors

- Edit `README.Rmd`, not `README.md`.
- `NAMESPACE` is generated by roxygen2.
- The current vignette set in `vignettes/` remains a primary long-form
  documentation source.
